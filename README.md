# Task Tracker CLI
A simple command-line tool to manage your tasks (add, list, update, delete, mark in-progress/done).

## Installation
Clone the repo

## Features
- Add tasks
- Update tasks
- Delete tasks
- Mark tasks as in-progress or done
- List tasks (all, todo, in-progress, done)

## running the script 
python3 task_cli.py  <command>  <argument>

## Storage
All tasks are saved in a `tasks.json` file in the same folder.
Each task has:
- id
- description
- status (todo, in-progress, done)
- createdAt
- updatedAt
